8th Swing %
This changes the swing of the 8th-note (the "and"). 0% places the 8th-note on the exact halfway point between beats. 100% places them a full 16th-note later (as if a dotted-8th/16th-note figure). 67% is a pure 8th-triplet swing. NOTE: 8th-note swing does not effect 16th-note placement. They are independent in Groove-O-Nome.
16th Swing %
This changes the swing of the 16th-notes (the "ee" and "uh"). 0% places the 16th-notes on exact fourths of the beat. 100% places them a full 32nd-note later (as if a dotted-16th/32nd-note figure). 67% is a pure 16th triplet swing. NOTE: 16th-note swing does not effect 8th-note placement. They are independent in Groove-O-Nome.
Random Mute %
This changes the percentage chance that the metronome will randomly mute itself. 0% means every note is played. 100% means no notes are played. This is great to test your internal sense of the time. Set it to a high percentage and see if you can still play along!
